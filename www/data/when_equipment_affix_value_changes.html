When Equipment Affix Value Changes

This event is triggered when Equipment Affix values change. The Owner of the Equipment will receive it. Configure this in the Item Node Graph

  • Equipment Owner GUID (GUID) -
  • Equipment Index (Integer) -
  • Affix ID (Integer) - The corresponding ID of this Entry within the Equipment Affixes
