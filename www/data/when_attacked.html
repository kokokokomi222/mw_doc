When Attacked

This event is triggered when the Entity is attacked (In Classic Mode, due to the Craftsperson's settings, the actual damage may differ from other scenarios.)

Output Values

  • Event Source Entity (Entity) -
  • Event Source GUID (GUID) -
  • Attacker Entity (Entity) -
  • Damage (Floating Point Numbers) - Actual damage dealt. If no damage is dealt due to Invincible or other reasons, the amount is 0
  • Attack Tag List (String List) -
  • Elemental Type (Enumeration) -
  • Elemental Attack Potency (Floating Point Numbers) - Elemental Gauge in the Attack
