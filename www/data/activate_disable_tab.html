Activate/Disable Tab

Edit the Tab state by ID in the Target Entity's Tab Component

Input Parameters

  • Target Entity (Entity) - Active Entity
  • Tab ID (Integer) - Identifier for the Tab
  • Activate (Boolean) - If set to True, it is active and can be selected
