Get Entity's Mini-Map Marker Status

Searches the configuration and activation status of the Entity's current Mini-map Marker

Input Parameters

  • Target Entity (Entity) - Runtime Entity

Output Values

  • Full Mini-Map Marker ID List (Integer List) - Complete list of Mini-map Marker IDs for this Entity
  • Active Mini-Map Marker ID List (Integer List) - Complete list of active Mini-map Marker IDs for this Entity
  • Inactive Mini-Map Marker ID List (Integer List) - Complete list of inactive Mini-map Marker IDs for this Entity
