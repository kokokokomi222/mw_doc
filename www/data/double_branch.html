Double Branch

Branches execution flow based on a boolean value.

Input Parameters

  • Condition (Boolean) - Boolean value that decides which path to take

Usage

You can execute different logic depending on a boolean condition with this node. In most other programming languages, this node would be called "if".

Example

If game_started is Yes, we are printing the value of game_mode. If game_started is No, we are printing "Game hasn't started".

Performance

This node took less than 1 unit to run on average.

Performance may vary for various reasons. These numbers are just for giving a sense of rough magnitude. Performance may differ from profiling in a test play when playing a published stage. First execution of a node within a node graph execution tend to take a bit longer.
