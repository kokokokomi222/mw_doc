Set UI Control (Group) Status

Edit the state of the UI Control in the Target Player's Interface Layout by its UI Control ID

Input Parameters

  • Target Player (Entity) - Active Player Entity
  • UI Control Index (Integer) - Identifier for the UI Control
  • Display Status (Enumeration) - Off: Invisible and logic not runningOn: Visible and logic running normallyHidden: Invisible and logic running normally
