# Summary
Branches execution flow based on a boolean value.

# Input Parameters
* [input:Condition] (Boolean) - Boolean value that decides which path to take

# Usage
You can execute different logic depending on a boolean condition with this node.
In most other programming languages, this node would be called "if".

# Example
If `game_started` is `Yes`, we are printing the value of `game_mode`.
If `game_started` is `No`, we are printing `"Game hasn't started"`.
