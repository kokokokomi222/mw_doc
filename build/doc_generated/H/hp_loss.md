# Summary
 Directly cause the specified target to lose HP. Losing HP is not an attack, so it does not trigger attack-related events

# Input Parameters
* [input:Target_Entity] (Entity) - Target that loses HP
* [input:HP_Loss] (Floating Point Numbers) - The amount of HP lost in this instance
* [input:Lethal] (Boolean) - If set to False, this HP loss will leave the Target with at least 1 HP remaining
* [input:Can_be_blocked_by_invincibility] (Boolean) - If set to True, and the Target is set to Invincible via Unit Status, HP loss has no effect
* [input:Can_be_Blocked_by_Locked_HP?] (Boolean) - If set to True, and the Target's HP is locked via Unit Status, HP loss has no effect
* [input:Damage_Pop-Up_Type] (Enumeration) - No Pop-UpNormal Pop-UpCRIT Hit Pop-Up
