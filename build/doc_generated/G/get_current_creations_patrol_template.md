# Summary
 Returns the Patrol Template information of the specified Creation Entity

# Input Parameters
* [input:Creation_Entity] (Entity) - Runtime Creation Entity

# Output Values
* [output:Patrol_Template_ID] (Integer) - The Patrol Template ID currently active on this Creation
* [output:Path_Index] (Integer) - The Path ID referenced by the Creation's currently active Patrol Template
* [output:Target_Waypoint_Index] (Integer) - The Waypoint ID the Creation will move to next
