# Summary
 When the Player Marker option is selected and a corresponding Player Entity is linked in the Node Graph, the Target Entity's display on the mini-map changes to that Player's avatar

# Input Parameters
* [input:Target_Entity] (Entity) - Entity that owns the Mini-map Marker component to be edited
* [input:Mini-Map_Marker_ID] (Integer) - ID of the specified Mini-map Marker to be edited
* [input:Corresponding_Player_Entity] (Entity) - Changes the avatar of the corresponding Player Entity
