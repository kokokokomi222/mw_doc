# Summary
 Set the value of the specified Custom Variable on the Target Entity

# Input Parameters
* [input:Target_Entity] (Entity) - The variable is mounted on this entity
* [input:Variable_Name] (String) - Custom variable name. Must be unique
* [input:Variable_Value] (Generic) - Value assigned to this variable
* [input:Trigger_Event] (Boolean) - Default: True. When set to False, this custom variable editing will not trigger the On Custom Variable Change event
