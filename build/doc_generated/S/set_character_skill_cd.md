# Summary
 Directly set the cooldown time of a specific skill slot on the target character to a specified value.

# Input Parameters
* [input:Target_Entity] (Entity) - Active Character Entity
* [input:Character_Skill_Slot] (Enumeration) - The Skill Slot to edited: Normal Attack, Skill 1-E, Skill 2-Q, Skill 3-R, Skill 4-T, or Custom Skill
* [input:CD_Time] (Floating Point Numbers) - Edited Cooldown will be set to this input value
* [input:Limit_Maximum_CD_Time] (Boolean) - If set to True, the edited Cooldown cannot be less than the specified minimum value
