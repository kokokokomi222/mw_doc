# Summary
 Set the value of the specified Node Graph Variable in the current Node Graph

# Input Parameters
* [input:Variable_Name] (String) - Name of the Node Graph Variable. Must be unique within the same Node Graph
* [input:Variable_Value] (Generic) - Value assigned to this variable
* [input:Trigger_Event] (Boolean) - Default: True. If set to False, this Node Graph Variable editing will not trigger the Variable Change Event
