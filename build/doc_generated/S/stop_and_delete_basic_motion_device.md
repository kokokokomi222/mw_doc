# Summary
 Stop and delete a running Motion Device

# Input Parameters
* [input:Target_Entity] (Entity) - Active Entity
* [input:Motion_Device_Name] (String) - Identifier for this motion device
* [input:Stop_All_Basic_Motion_Devices] (Boolean) - If set to True, stops all Basic Motion Devices on this Entity. If set to False, stops only the Motion Device whose name matches the specified Motion Device
