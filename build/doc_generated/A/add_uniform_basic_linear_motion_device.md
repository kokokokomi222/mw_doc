# Summary
 Dynamically add a Basic Motion Device with Uniform Linear Motion at runtime

# Input Parameters
* [input:Target_Entity] (Entity) - Active Entity
* [input:Motion_Device_Name] (String) - Identifier for this motion device
* [input:Motion_Device_Duration] (Floating Point Numbers) - The duration for which this motion device remains active
* [input:Velocity_Vector] (3D Vector) - Determines the magnitude and direction of the velocity
