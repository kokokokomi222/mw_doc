# Summary
 Play a Timed Effect relative to the Target Entity. A valid Target Entity and Attachment Point are required

# Input Parameters
* [input:Special_Effects_Asset] (Config ID) - Identifier for this Effect
* [input:Target_Entity] (Entity) - If the Entity does not exist, the Effect will not play
* [input:Attachment_Point_Name] (String) - If the Attachment Point Name does not exist, the Special Effect will not play
* [input:Move_With_the_Target] (Boolean) - If set to True, follows the Target Entity's Motion
* [input:Rotate_With_the_Target] (Boolean) - If set to True, follows the Target Entity's Rotation
* [input:Location_Offset] (3D Vector) - Location Offset relative to the Target Entity's specified Attachment Point
* [input:Rotation_Offset] (3D Vector) - Rotation offset relative to the Target Entity's specified Attachment Point
* [input:Zoom_Multiplier] (Floating Point Numbers) - The Zoom Multiplier of this Effect
* [input:Play_Built-In_Sound_Effect] (Boolean) - If set to True, plays the built-in Sound Effect as well
