# Summary
 Create a Projectile Entity using the Prefab ID. This function is similar to \[Create Prefab\], but includes an additional \[Track Target\] parameter, which sets the tracking target for projectiles of the Tracking type in the Projectile Motion Device Component of the created Entity

# Input Parameters
* [input:Prefab_ID] (Prefab ID) - Identifier for this Projectile Prefab
* [input:Location] (3D Vector) - Absolute Location
* [input:Rotate] (3D Vector) - Absolute Rotation
* [input:Owner_Entity] (Entity) - Determines whether the created entity belongs to another entity
* [input:Track_Target] (Entity) - The Tracking Target set by the Tracking Projectile type in the Projectile Motion Device component
* [input:Overwrite_Level] (Boolean) - When set to False, the \[Level\] parameter has no effect
* [input:Level] (Integer) - Determines the Level when the entity is created
* [input:Unit_Tag_Index_List] (Integer List) - Determines the Unit Tags carried when this entity is created

# Output Values
* [output:Created_Entity] (Entity) - This Entity inherits the attributes of the Projectile Prefab
