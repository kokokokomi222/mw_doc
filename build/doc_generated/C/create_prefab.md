# Summary
 Create an Entity by Prefab ID

# Input Parameters
* [input:Prefab_ID] (Prefab ID) - Identifier for this Prefab
* [input:Location] (3D Vector) - Absolute Location
* [input:Rotate] (3D Vector) - Absolute Rotation
* [input:Owner_Entity] (Entity) - Determines whether the created entity belongs to another entity
* [input:Overwrite_Level] (Boolean) - When set to False, the \[Level\] parameter has no effect
* [input:Level] (Integer) - Determines the Level when the entity is created
* [input:Unit_Tag_Index_List] (Integer List) - Determines the Unit Tags carried when this entity is created

# Output Values
* [output:Created_Entity] (Entity) - Entities created in this way do not have a GUID
