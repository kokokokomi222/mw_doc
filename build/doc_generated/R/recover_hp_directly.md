# Summary
 Directly restore HP to the specified Target Entity. Unlike \[Recover HP\], this node does not require an Ability Unit

# Input Parameters
* [input:Recover_Initiator_Entity] (Entity) - The Entity that initiates healing
* [input:Recover_Target_Entity] (Entity) - The Target Entity to be healed
* [input:Recovery_Amount] (Floating Point Numbers) - The amount of HP restored in this healing instance
* [input:Ignore_Recovery_Amount_Adjustment] (Boolean) - If set to True, this healing amount is not affected by the Target's Unit Status effects that adjust healing
* [input:Aggro_Generation_Multiplier] (Floating Point Numbers) - The Aggro generated by this healing, expressed as a multiplier. Only applicable when using Custom Aggro Mode
* [input:Aggro_Generation_Increment] (Floating Point Numbers) - The Aggro generated by this healing, expressed as an incremental value. Only applicable when using Custom Aggro Mode
* [input:Healing_Tag_List] (String List) - The list of tags associated with this healing action. These can be accessed in theWhen HP Is RecoveredandWhen Initiating HP Recoveryevents to identify a specific healing action
