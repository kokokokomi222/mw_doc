# Summary
 This event is triggered when the Custom Variable of the Entity associated with the current Node Graph changes The previous and current values are Generic. Determine the Generic type before you can correctly receive events for Custom Variables of the corresponding type Vessel-type Custom Variables do not provide before-value and after-value Output Parameters

# Output Values
* [output:Event_Source_Entity] (Entity) - The Entity associated with this Node Graph
* [output:Event_Source_GUID] (GUID) - GUID of the Entity associated with this Node Graph
* [output:Variable_Name] (String) - Name of the Variable that was changed
* [output:Pre-Change_Value] (Generic) - The Variable's value before the change
* [output:Post-Change_Value] (Generic) - The Variable's value after the change
