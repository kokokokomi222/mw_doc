# Summary
 This event is triggered when a Unit Status is removed for any reason or when its Runtime Duration expires

# Output Values
* [output:Event_Source_Entity] (Entity) - 
* [output:Event_Source_GUID] (GUID) - 
* [output:Unit_Status_Config_ID] (Config ID) - 
* [output:Applier_Entity] (Entity) - 
* [output:Infinite_Duration] (Boolean) - 
* [output:Remaining_Status_Duration] (Floating Point Numbers) - 
* [output:Remaining_Status_Stacks] (Integer) - 
* [output:Remover_Entity] (Entity) - 
* [output:Removal_Reason] (Enumeration) - Status Replacement: The Unit Status was removed because it was replaced by another statusDuration Exceeded: The Unit Status exceeded its runtime durationDispelled: The Unit Status was removed directlyStatus Expired: The Unit Status became invalid due to other reasonsClass Changed: The Unit Status was removed due to a class change
* [output:Slot_ID] (Integer) - ID of the Unit Status slot that changed
