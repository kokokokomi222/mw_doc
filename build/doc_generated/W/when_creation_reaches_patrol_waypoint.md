# Summary
 When the Send Node Graph Event on Arrival option is enabled for a waypoint in the Patrol template, a Node Graph Event is triggered once the specified conditions are met This Node Graph Event can only be received by the creation's node graph

# Output Values
* [output:Creation_Entity] (Entity) - Runtime Creation Entity
* [output:Creation_GUID] (GUID) - The GUID of the Creation. If it was not an initially placed Creation, the output is empty
* [output:Current_Patrol_Template_ID] (Integer) - The Patrol Template ID currently active on this Creation
* [output:Current_Path_Index] (Integer) - The Path ID referenced by the Creation's currently active Patrol Template
* [output:Current_Reached_Waypoint_ID] (Integer) - The Waypoint ID the Creation has currently reached
* [output:Next_Waypoint_ID] (Integer) - The Waypoint ID the Creation will move to next
