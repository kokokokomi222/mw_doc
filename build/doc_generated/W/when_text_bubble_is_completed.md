# Summary
 This event can only be mounted by Text Bubble Components and is received by the Entity's Node Graph that completed the dialogue Completion refers to when the final line of dialogue has finished playing

# Output Values
* [output:Bubble_Owner_Entity] (Entity) - Runtime Entity with the Text Bubble component mounted
* [output:Character_Entity] (Entity) - Target Character of the current Bubble dialogue
* [output:Text_Bubble_Configuration_ID] (Config ID) - Currently active Text Bubble Config ID
* [output:Text_Bubble_Completion_Count] (Integer) - Number of times the currently active Text Bubble has been fully played for this dialogue Character
