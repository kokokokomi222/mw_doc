# Summary
 This event triggers when objects and creations within the stage are destroyed. This event can only trigger on stage entities.

* [output:Event_Source_GUID] (GUID) - 
* [output:Location] (3D Vector) - 
* [output:Orientation] (3D Vector) - 
* [output:Entity_Type] (Enumeration) - 
* [output:Faction] (Faction) - 
* [output:Damage_Source] (Entity) - 
* [output:Owner_Entity] (Entity) - 
* [output:Custom_Variable_Component_Snapshot] (Custom Variable Snapshot) - On destroy, captures a snapshot of the Custom Variable component on this Entity. Use the Search Custom Variable Snapshot node to retrieve its Custom Variable values
