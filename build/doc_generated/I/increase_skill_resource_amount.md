# Summary
 Modifying the resource amount of a skill will add an increase to the current value; this increase can be a negative number.

# Input Parameters
* [input:Target_Entity] (Entity) - Active Character Entity
* [input:Skill_Resource_Config_ID] (Config ID) - Skill Resource Identifier
* [input:Increase_Value] (Floating Point Numbers) - The modified value is: original value + increase value.
