Get Aggro List of Creation in Default Mode

Returns the Aggro List in Classic Mode. This Node only outputs a valid list when the Aggro Configuration is set to [Default Type]

Input Parameters

  • Creation Entity (Entity) - Runtime Creation Entity

Output Values

  • Aggro List (Entity List) - Unordered list of Entities this Creation currently has Aggro against
