When Unit Status Ends

This event is triggered when a Unit Status is removed for any reason or when its Runtime Duration expires

Output Values

  • Event Source Entity (Entity) -
  • Event Source GUID (GUID) -
  • Unit Status Config ID (Config ID) -
  • Applier Entity (Entity) -
  • Infinite Duration (Boolean) -
  • Remaining Status Duration (Floating Point Numbers) -
  • Remaining Status Stacks (Integer) -
  • Remover Entity (Entity) -
  • Removal Reason (Enumeration) - Status Replacement: The Unit Status was removed because it was replaced by another statusDuration Exceeded: The Unit Status exceeded its runtime durationDispelled: The Unit Status was removed directlyStatus Expired: The Unit Status became invalid due to other reasonsClass Changed: The Unit Status was removed due to a class change
  • Slot ID (Integer) - ID of the Unit Status slot that changed
