When Unit Status Changes

This event is triggered when the Stack Count of a Unit Status changes This event is triggered when Unit Status effects are applied or removed

Output Values

  • Event Source Entity (Entity) -
  • Event Source GUID (GUID) -
  • Unit Status Config ID (Config ID) -
  • Applier Entity (Entity) -
  • Infinite Duration (Boolean) -
  • Remaining Status Duration (Floating Point Numbers) -
  • Remaining Status Stacks (Integer) - Edited Stack Count
  • Original Status Stacks (Integer) - Previous Stack Count
  • Slot ID (Integer) - ID of the Unit Status slot that changed
