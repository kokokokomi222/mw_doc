When Text Bubble Is Completed

This event can only be mounted by Text Bubble Components and is received by the Entity's Node Graph that completed the dialogue Completion refers to when the final line of dialogue has finished playing

Output Values

  • Bubble Owner Entity (Entity) - Runtime Entity with the Text Bubble component mounted
  • Character Entity (Entity) - Target Character of the current Bubble dialogue
  • Text Bubble Configuration ID (Config ID) - Currently active Text Bubble Config ID
  • Text Bubble Completion Count (Integer) - Number of times the currently active Text Bubble has been fully played for this dialogue Character
