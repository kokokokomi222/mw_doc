Add Affix to Equipment

Add a preconfigured Affix to the specified Equipment instance, with the option to overwrite the Affix value

Input Parameters

  • Equipment ID (Integer) - Integer ID generated during Equipment Initialization to identify the equipment instance
  • Affix Config ID (Config ID) - The Config ID of the preconfigured Affix defined in Equipment Data Management
  • Overwrite Affix Value (Boolean) -
  • Affix Value (Floating Point Numbers) - Can overwrite the value on a pre-configured Affix
