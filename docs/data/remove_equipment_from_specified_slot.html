Remove Equipment from Specified Slot

This documentation was published soon after the release of this node. Some of the information may be inaccurate. Let me know if you find something wrong.

Remove equipment from slot.

Input Parameters

  • Equipment Slot Owner Entity (Entity) - Character entity to remove the equipment from.
  • Equipment Slot Row Count (Integer) - Row number of the equipment slot to remove from. This is 0-indexed.
  • Equipment Slot Column Count (Integer) - Column number of the equipment slot to remove from. This is 0-indexed.

Usage

Use this node to remove equipment from an equipment slot. Removed equipment stays in the inventory.

Example

In this example, we remove the equipment at second row (indexed by 1) and third column (indexed by 2). This is the result of removal. Removed equipment stays in the inventory.

Notes

  • If Equipment Slot Owner Entity is not a character entity, this node raises "Entity component does not exist" error, but continues the execution.
  • If Equipment Slot Owner Entity does not have an "Equipment Slot" component, this node raises "Entity component does not exist" error, but continues the execution.
  • If there is no equipment slot at the coordinate indicated by Equipment Slot Row Count and Equipment Slot Column Count, this node is no-op.
  • If there is no equipment at the slot, this node is no-op.

Performance

When equipment was removed, this node took ~10 units to run on average. If there is no equipment at the slot, this node took ~2 units to run on average.

Performance may vary for various reasons. These numbers are just for giving a sense of rough magnitude. Performance may differ from profiling in a test play when playing a published stage. First execution of a node within a node graph execution tend to take a bit longer.
