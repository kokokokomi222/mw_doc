Create Projectile

Create a Projectile Entity using the Prefab ID. This function is similar to [Create Prefab], but includes an additional [Track Target] parameter, which sets the tracking target for projectiles of the Tracking type in the Projectile Motion Device Component of the created Entity

Input Parameters

  • Prefab ID (Prefab ID) - Identifier for this Projectile Prefab
  • Location (3D Vector) - Absolute Location
  • Rotate (3D Vector) - Absolute Rotation
  • Owner Entity (Entity) - Determines whether the created entity belongs to another entity
  • Track Target (Entity) - The Tracking Target set by the Tracking Projectile type in the Projectile Motion Device component
  • Overwrite Level (Boolean) - When set to False, the [Level] parameter has no effect
  • Level (Integer) - Determines the Level when the entity is created
  • Unit Tag Index List (Integer List) - Determines the Unit Tags carried when this entity is created

Output Values

  • Created Entity (Entity) - This Entity inherits the attributes of the Projectile Prefab
