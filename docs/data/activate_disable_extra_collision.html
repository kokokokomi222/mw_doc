Activate/Disable Extra Collision

Edit data in the Entity's Extra Collision Component to enable/disable Extra Collision

Input Parameters

  • Target Entity (Entity) - Active Entity
  • Extra Collision ID (Integer) - Identifier for this Extra Collision
  • Activate (Boolean) - Set to True to activate
