Mount Looping Special Effect

Mount a Looping Effect relative to the Target Entity. A valid Target Entity and Attachment Point are required This node returns an Effect Instance ID that can be stored. When using the [Clear Looping Effects] node later, use this Effect Instance ID to clear the specified Looping Effect

Input Parameters

  • Special Effects Asset (Config ID) - Identifier for this Effect
  • Target Entity (Entity) - If the Entity does not exist, the Effect will not play
  • Attachment Point Name (String) - If the Attachment Point Name does not exist, the Special Effect will not play
  • Move With the Target (Boolean) - If set to True, follows the Target Entity's Motion
  • Rotate With the Target (Boolean) - If set to True, follows the Target Entity's Rotation
  • Location Offset (3D Vector) - Location Offset relative to the Target Entity's specified Attachment Point
  • Rotation Offset (3D Vector) - Rotation offset relative to the Target Entity's specified Attachment Point
  • Zoom Multiplier (Floating Point Numbers) - The Zoom Multiplier of this Effect

Output Values

  • Special Effect Instance ID (Integer) - The Instance ID automatically generated when mounting this Effect
