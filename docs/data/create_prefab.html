Create Prefab

Creates a prefab.

Input Parameters

  • Prefab ID (Prefab ID) - Prefab ID for the prefab to create. Typically copied from the top of the prefab details panel. e.g. 1012345678
  • Location (3D Vector) - Location in world coordinate space.
  • Rotate (3D Vector) - Rotation in world coordinate space.
  • Owner Entity (Entity) - Owner of the created entity. Can be left empty.
  • Overwrite Level (Boolean) - If No, Level is ignored and the prefab setting is used.
  • Level (Integer) - Level of the created entity. Only used if Overwrite Level is Yes. Must be between 1 and 200, inclusive.
  • Unit Tag Index List (Integer List) - List of unit tag IDs to put on the created entity. e.g. [1023456789, 1023456790]. Can be left empty. See "Notes" section below for detail.

Output Values

  • Created Entity (Entity) - The created entity.

Usage

Use this node to create an instance of a prefab. This is the most straightforward way to dynamically create an entity.

If you want to create an entity just once with static position, rotation, and level, you can use Create Entity instead.

Notes

  • If prefab with Prefab ID does not exist, this node raises a "Prefab ID does not exist" error, but continues the execution.
  • If Location is outside the stage's "Effective Scene Range", this node is no-op. No entity is created.
  • If Owner Entity is set to an entity that does not exist or is removed, this node still works and creates an entity but sets the owner of the entity to none.
  • If Overwrite Level is Yes and Level is less than 1 or greater than 200, this node is no-op. No entity is created. If Overwrite Level is No, any value of Level is fine and the node will successfully execute.
  • If you set unit tags in the prefab detail panel, those tags will be overwritten by Unit Tag Index List. For example, if tag A and B were set in the prefab detail panel, but tag B and C were passed into Unit Tag Index List, then the created entity has tag B and C. However, if Unit Tag Index List is left empty, or an empty list is passed into it, the tags set in the prefab detail panel will be used.
  • Entities created by this node do not have a GUID. If Query GUID by Entity is called on an entity created by this node, it returns 0.

Performance

Creating an entity with default prefab setting took ~70 units to run on average. Setting Unit Tag Index List does not seem to visibly affect the performance of this node.

Performance may vary for various reasons. These numbers are just for giving a sense of rough magnitude. Performance may differ from profiling in a test play when playing a published stage. First execution of a node within a node graph execution tend to take a bit longer.
