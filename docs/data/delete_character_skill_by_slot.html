Delete Character Skill by Slot

Delete the skill in the specified slot of the Target Character

Input Parameters

  • Target Entity (Entity) - Active Character Entity
  • Character Skill Slot (Enumeration) - The Skill Slot to be deleted: Normal Attack, Skill 1-E, Skill 2-Q, Skill 3-R, Skill 4-T, or Custom Skill
