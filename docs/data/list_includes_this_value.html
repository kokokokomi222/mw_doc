List Includes This Value

Returns whether a list contains a value.

Input Parameters

  • List (Generic) - List to search in.
  • Value (Generic) - Value to search for.

Output Values

  • Include (Boolean) - Yes if the value is in the list, No otherwise.

Usage

Use this node to test whether a certain value is in a list. If you also need to know at which index the value is at, use Search List and Return Value ID instead.

Example

In this example, we have a string list ["Kokomi", "Furina", "Yelan", "Escoffier"]. We check whether "Furina" is in this list, then print "Furina is in the team!" if true.

Notes

  • Changing the type of List or Value will change the type of the other to match it.
  • If List is left empty, this node is guaranteed to return No regardless of Value.
  • For floating point numbers, this node uses approximate equality (e.g. 10.0 and 10.00001 is considered equal by this node). This approximate equality seems to be same as Equal, but we cannot be completely sure.

Performance

For an integer list of length 10, this node took ~1 units to run on average.

For an integer list of length 1000, this node took ~6 units to run on average. When the value was located at index 0 of the list, performance does not seem to be affected visibly.

Performance may vary for various reasons. These numbers are just for giving a sense of rough magnitude. Performance may differ from profiling in a test play when playing a published stage. First execution of a node within a node graph execution tend to take a bit longer.
