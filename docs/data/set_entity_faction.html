Set Entity Faction

This server node is only available in Beyond Mode.

Sets the faction of an entity.

Input Parameters

  • Target Entity (Entity) - Entity to set the faction.
  • Faction (Faction) - Faction that the entity will change into.

Usage

Faction is primarily used to split entities into teams with varying friendly/hostile relationships. Creatures are only aggroed by hostile entities, and ability units can be filtered to only target hostile faction entities (which is the default).

Factions can also be used for grouping players into teams for settlement. By rewarding the result and ranking to factions, faction players can be motivated to work together and compete against other factions.

Factions can be configured in "Stage Settings" but only in Beyond Mode. The faction of each entity can be individually configured in the entity details panel.

Notes

  • If Target Entity does not exist, this node raises "Entity does not exist" error, but continues the execution.
  • If Faction is not an ID for a faction that exists, this node is no-op and When Entity Faction Changes does not trigger.
  • If Target Entity is the stage entity, this node is no-op and When Entity Faction Changes does not trigger. The stage does not have a faction, and it cannot be set.
  • If Target Entity is a player entity, then the player entity and all character entities owned by the player change the faction to Faction and When Entity Faction Changes triggers on all changed entities.
  • If Target Entity is a character entity, then the player entity that owns the character and all character entities owned by that player change the faction to Faction and When Entity Faction Changes triggers on all changed entities.
  • If Faction is the same as the current faction of the entity, it still triggers When Entity Faction Changes.

Performance

This node took ~5 units to run on average.

Performance may vary for various reasons. These numbers are just for giving a sense of rough magnitude. Performance may differ from profiling in a test play when playing a published stage. First execution of a node within a node graph execution tend to take a bit longer.
