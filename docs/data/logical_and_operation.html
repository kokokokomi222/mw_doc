Logical AND Operation

Performs boolean AND.

Input Parameters

  • Input 1 (Boolean) - First operand.
  • Input 2 (Boolean) - Second operand.

Output Values

  • Result (Boolean) - Result of boolean AND of the two input values.

Usage

The output of the node is Yes, if and only if both of the inputs are Yes.
Input 1 Input 2 Result
Yes Yes Yes
Yes No No
No Yes No
No No No

Notes

  • If you leave any of the inputs empty, the input is considered to be No.

Performance

This node took ~2 units to run on average.

Performance may vary for various reasons. These numbers are just for giving a sense of rough magnitude. Performance may differ from profiling in a test play when playing a published stage. First execution of a node within a node graph execution tend to take a bit longer.
