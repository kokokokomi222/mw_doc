Switch Follow Motion Device Target by Entity

Switch the Follow Target of the Follow Motion Device by Entity

Input Parameters

  • Target Entity (Entity) - Entity that a Follow Motion Device is attached to
  • Follow Target Entity (Entity) - The Entity that follows the Target
  • Follow Target Attachment Point Name (String) - Name of the Attachment Point to follow
  • Location Offset (3D Vector) - Location Offset based on the Follow Coordinate System
  • Rotation Offset (3D Vector) - Rotation Offset based on the Follow Coordinate System
  • Follow Coordinate System (Enumeration) - Options: Relative Coordinate System or World Coordinate System
  • Follow Type (Enumeration) - Options: Completely Follow, Follow Location, Follow Rotation
