When Creation Reaches Patrol Waypoint

When the Send Node Graph Event on Arrival option is enabled for a waypoint in the Patrol template, a Node Graph Event is triggered once the specified conditions are met This Node Graph Event can only be received by the creation's node graph

Output Values

  • Creation Entity (Entity) - Runtime Creation Entity
  • Creation GUID (GUID) - The GUID of the Creation. If it was not an initially placed Creation, the output is empty
  • Current Patrol Template ID (Integer) - The Patrol Template ID currently active on this Creation
  • Current Path Index (Integer) - The Path ID referenced by the Creation's currently active Patrol Template
  • Current Reached Waypoint ID (Integer) - The Waypoint ID the Creation has currently reached
  • Next Waypoint ID (Integer) - The Waypoint ID the Creation will move to next
