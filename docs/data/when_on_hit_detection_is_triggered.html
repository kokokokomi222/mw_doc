When On-Hit Detection Is Triggered

This event is triggered when the On-Hit Detection Component's Owner hits other Entities or the Scene

Output Values

  • Event Source Entity (Entity) -
  • Event Source GUID (GUID) -
  • On-Hit Hurtbox (Boolean) - If set to False: The environment was hitIf set to True: An Entity was hit. Retrieve values from the Hit Entity output parameter
  • On-Hit Entity (Entity) - Hit Entity is only valid when a Hurtbox is hit
  • On-Hit Location (3D Vector) -
