Set Player Markers on the Mini-Map

When the Player Marker option is selected and a corresponding Player Entity is linked in the Node Graph, the Target Entity's display on the mini-map changes to that Player's avatar

Input Parameters

  • Target Entity (Entity) - Entity that owns the Mini-map Marker component to be edited
  • Mini-Map Marker ID (Integer) - ID of the specified Mini-map Marker to be edited
  • Corresponding Player Entity (Entity) - Changes the avatar of the corresponding Player Entity
