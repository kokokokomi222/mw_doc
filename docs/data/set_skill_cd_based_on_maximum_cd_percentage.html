Set Skill CD Based on Maximum CD Percentage

Modify the skill in a character's skill slot by adjusting the percentage of the skill's maximum cooldown time.

Input Parameters

  • Target Entity (Entity) - Active Character Entity
  • Character Skill Slot (Enumeration) - The Skill Slot to edited: Normal Attack, Skill 1-E, Skill 2-Q, Skill 3-R, Skill 4-T, or Custom Skill
  • Ratio Value (Floating Point Numbers) - The revised actual cooldown time is: original cooldown time * ratio value
  • Limit Maximum CD Time (Boolean) - If set to True, the edited Cooldown cannot be less than the specified minimum value
