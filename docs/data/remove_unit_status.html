Remove Unit Status

Remove a specified Unit Status from the Target Entity. Either all stacks or a single stack can be removed

Input Parameters

  • Remove Target Entity (Entity) - The Entity from which the Unit Status will be removed
  • Unit Status Config ID (Config ID) - Identifier for this Unit Status
  • Removal Method (Enumeration) - All Coexisting Statuses with the Same Name: Removes all statuses applied with this Config ID that share the same nameStatus With Fastest Stack Loss: Removes one stack from the status that loses stacks the fastest
  • Remover Entity (Entity) - Determines the Remover Entity for this action. Defaults to the Entity associated with this Node Graph
