Recover HP Directly

Directly restore HP to the specified Target Entity. Unlike [Recover HP], this node does not require an Ability Unit

Input Parameters

  • Recover Initiator Entity (Entity) - The Entity that initiates healing
  • Recover Target Entity (Entity) - The Target Entity to be healed
  • Recovery Amount (Floating Point Numbers) - The amount of HP restored in this healing instance
  • Ignore Recovery Amount Adjustment (Boolean) - If set to True, this healing amount is not affected by the Target's Unit Status effects that adjust healing
  • Aggro Generation Multiplier (Floating Point Numbers) - The Aggro generated by this healing, expressed as a multiplier. Only applicable when using Custom Aggro Mode
  • Aggro Generation Increment (Floating Point Numbers) - The Aggro generated by this healing, expressed as an incremental value. Only applicable when using Custom Aggro Mode
  • Healing Tag List (String List) - The list of tags associated with this healing action. These can be accessed in theWhen HP Is RecoveredandWhen Initiating HP Recoveryevents to identify a specific healing action
