Split 3D Vector

Outputs components of a 3D vector.

Input Parameters

  • 3D Vector (3D Vector) - Input 3D vector

Output Values

  • X-Component (Floating Point Numbers) - X component of the input vector
  • Y-Component (Floating Point Numbers) - Y component of the input vector
  • Z-Component (Floating Point Numbers) - Z component of the input vector

Usage

Use this node to get the components of a 3D vector.

Example

In this example, we get the location of the character entity that selected the tab. We get the y component of that entity location and print it.

Performance

This node took ~2 units to run on average when only one of the output values were used. If you use all 3 output values, it can take ~4 units on average.

Performance may vary for various reasons. These numbers are just for giving a sense of rough magnitude. Performance may differ from profiling in a test play when playing a published stage. First execution of a node within a node graph execution tend to take a bit longer.
