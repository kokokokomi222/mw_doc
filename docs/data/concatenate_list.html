Concatenate List

Appends a list to the end of another list.

Input Parameters

  • Target List (Generic List) - List to append to. This list will be modified.
  • Input List (Generic List) - List to append.

Usage

Use this node to append a list to the end of another list. For example, if Target List is [1, 2, 3], and Input List is [99, 100], Target List will be modified to [1, 2, 3, 99, 100].

Example

In this example, a local variable was initially set to ["Venti", "Zhongli", "Raiden"]. A second list, ["Nahida", "Furina", "Mavuika"], was appended to the end of the first list. The first list is modified to ["Venti", "Zhongli", "Raiden", "Nahida", "Furina", "Mavuika"] after this node graph runs.

Notes

  • The maximum size of a list is 1000. If the resulting list would have the length greater than 1000, this node is a no-op. It will NOT partially append until 1000 entries are filled.
  • Changing the type of Target List or Input List will change the type of the other to match it.
  • Passing the same reference of a list to both Target List and Input List works as expected, doubling the target list's size.

Performance

Concatenating a list of length 500 with another list of length 500 took 7 units to run on average.

Performance may vary for various reasons. These numbers are just for giving a sense of rough magnitude. Performance may differ from profiling in a test play when playing a published stage. First execution of a node within a node graph execution tend to take a bit longer.
