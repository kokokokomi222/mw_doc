Recover HP

Restore HP to the specified Target Entity via an Ability Unit

Input Parameters

  • Target Entity (Entity) - Target of HP restoration
  • Recovery Amount (Floating Point Numbers) - The amount of HP restored in this healing instance
  • Ability Unit (String) - Referenced Ability Unit. Must be configured on the entity associated with this Node Graph
  • Overwrite Ability Unit Config (Boolean) - When set to True, the Recovery Amount overwrites the parameter of the same name in the Ability Unit. When set to False, the Ability Unit's original configuration is used
  • Recover Initiator Entity (Entity) - Determines the Initiator Entity of this healing action. Defaults to the Entity associated with this Node Graph. Affects healer identification in events such asWhen HP Is RecoveredandWhen Initiating HP Recovery
