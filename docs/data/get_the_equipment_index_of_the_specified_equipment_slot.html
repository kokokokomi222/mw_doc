Get the Equipment Index of the Specified Equipment Slot

Get the equipment ID equipped at an equipment slot.

Input Parameters

  • Target Entity (Entity) - Character entity that has an "Equipment Slot" component.
  • Row (Integer) - Row number of the equipment slot to query. This is 0-indexed.
  • Column (Integer) - Column number of the equipment slot to query. This is 0-indexed.

Output Values

  • Equipment Index (Integer) - Equipment ID of the equipment equipped at the equipment slot. 0 if the slot is empty.

Usage

Use this node to get the equipment ID of the equipment, which is equipped at an equipment slot.

Notes

  • If Target Entity is not a character entity, this node raises "Entity component does not exist" error and returns 0.
  • If Target Entity does not have an "Equipment Slot" component, this node raises "Entity component does not exist" error and returns 0.
  • If there is no equipment slot at the coordinate indicated by Row and Column, this node returns 0.
