Get Self Entity

Returns the self entity.

Output Values

  • Self Entity (Entity) - Self entity, the entity that is mounting the node graph that is currently executing.

Usage

All node graphs must be mounted to an entity for it to run. When a node graph runs, it runs in the context of an entity. This node returns that entity.

Many event nodes provide Event Source Entity, which is typically the same as the self entity. However, this node may still be of use for a few reasons:

  • If you want two or more event nodes to share a logic, then this node can be used. See the "Example" section below for an example.
  • If you forwarded an event with Forwarding Event, Event Source Entity can differ from the self entity.
  • This node can help organize the node graph, because we don't have to rely on one Event Source Entity output pin to be connected to everything.
  • This node is shorter as a name for input.

Performance

This node took less than 1 unit to run on average.

Performance may vary for various reasons. These numbers are just for giving a sense of rough magnitude. Performance may differ from profiling in a test play when playing a published stage. First execution of a node within a node graph execution tend to take a bit longer.
