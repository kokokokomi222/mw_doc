Activate/Disable Collision Trigger

Edit the Collision Trigger Component data to Activate/Disable the Trigger at the specified ID

Input Parameters

  • Target Entity (Entity) - Active Entity
  • Trigger ID (Integer) - Identifier for this Collision Trigger
  • Activate (Boolean) - Set to True to activate
