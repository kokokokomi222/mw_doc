Add New Item to Custom Shop Sales List

Add items to the Custom Shop Sales List. Upon success, an Integer ID is generated in the Output Parameter as the item identifier

Input Parameters

  • Shop Owner Entity (Entity) -
  • Shop ID (Integer) - The Shop ID corresponding to the Shop component on the Shop Owner Entity
  • Shop Item Config ID (Config ID) -
  • Sell Currency Dictionary (Dictionary) -
  • Affiliated Tab ID (Integer) - 1 Equipment, 2 Consumables, 3 Materials, 4 Valuables
  • Limit Purchase (Boolean) -
  • Purchase Limit (Integer) -
  • Sort Priority (Integer) -
  • Can Be Sold (Boolean) -

Output Values

  • Item Index (Integer) -
