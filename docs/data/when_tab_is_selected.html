When Tab Is Selected

When the active tab is selected, it will send an event to the node graph The Entity Node Graph configured by the Tab Component will receive this event

Output Values

  • Event Source Entity (Entity) - Entity with the tab component mounted
  • Event Source GUID (GUID) - GUID of the Entity with the tab component mounted; outputs 0 if none exists
  • Tab ID (Integer) - ID of the tab
  • Selector Entity (Entity) - Character Entity that triggers the tab
