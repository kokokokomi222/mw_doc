When Entering Collision Trigger

The "Collision Trigger Source" range of a runtime entity A enters the "Collision Trigger" range of another runtime entity B Node graph events will be sent to the entity B configured with "Collision Trigger"

Output Values

  • Entering Entity (Entity) - Entity A (referenced above)
  • Entering Entity GUID (GUID) -
  • Trigger Entity (Entity) - Entity B (mentioned above)
  • Trigger Entity GUID (GUID) -
  • Trigger ID (Integer) - The trigger with the corresponding ID in Entity B's Collision Trigger Component
