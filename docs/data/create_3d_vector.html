Create 3D Vector

Creates a 3D vector.

Input Parameters

  • X-Component (Floating Point Numbers) - X-component of the new 3D vector.
  • Y-Component (Floating Point Numbers) - Y-component of the new 3D vector.
  • Z-Component (Floating Point Numbers) - Z-component of the new 3D vector.

Output Values

  • 3D Vector (3D Vector) - Created 3D vector.

Usage

Use this node to create a 3D vector.

Example

In this example, a prefab is created when the tab is selected. The location of the new prefab is randomly picked so that the x-component is between -20 and 20, the y-component is always 1, and the z-component is between -20 and 20.

Notes

  • If you leave any of the input empty, it defaults to 0.0.

Performance

This node took ~2 units to run on average.

Performance may vary for various reasons. These numbers are just for giving a sense of rough magnitude. Performance may differ from profiling in a test play when playing a published stage. First execution of a node within a node graph execution tend to take a bit longer.
