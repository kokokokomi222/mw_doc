Set UI Control (Group) Status

Changes the status of UI control.

Input Parameters

  • Target Player (Entity) - Entity of player to change the status of UI control for.
  • UI Control Index (Integer) - ID of the UI Control.
  • Display Status (Enumeration) - One of Off, On, Hidden.

Usage

UI controls are user interface elements such as text, button, pop-up, or progress bar. UI controls can be created and configured in the "Manage UI Control Groups" panel, which can be opened from the main menu. This node lets you modify the status of the UI control, making it visible or invisible.
