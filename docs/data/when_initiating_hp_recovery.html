When Initiating HP Recovery

This event is triggered on the initiating Entity when an Entity restores HP to other Entities

Output Values

  • Event Source Entity (Entity) -
  • Event Source GUID (GUID) -
  • Recover Target Entity (Entity) -
  • Recovery Amount (Floating Point Numbers) - Actual healing amount. If the Target Entity had not lost any HP prior to healing, the amount is 0
  • Recover Tag List (String List) -
