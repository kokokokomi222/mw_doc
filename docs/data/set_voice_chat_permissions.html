Set Voice Chat Permissions

Set the target player's permissions to use voice chat in the channel list

Input Parameters

  • Target Player (Entity) -
  • Channel List (Integer List) -
  • Restrict Voice Chat (Speak) (Boolean) - When enabled, other players cannot hear the target player's voice
  • Restrict Voice Chat (Listen) (Boolean) - When enabled, the target player cannot hear other players' voices
