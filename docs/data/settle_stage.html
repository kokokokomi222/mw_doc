Settle Stage

Settles the stage.

Usage

Settling the stage terminates the gameplay for every player, and all players are sent to the results screen that shows new achievements, competitive rank changes, scores and rankings of players, etc.

To actually set the results of settlement, use "Stage Settlement" execution nodes such as Set Player Settlement Success Status or Set Player Settlement Ranking Value.

In this example, the player who selected the tab will have their settlement status set to Victory. Then we settle the stage, ending the stage immediately.

Notes

  • This node removes all players and characters. Thus, no settlement result can be set after this node has been executed. We recommend that you do not run any logic after this node is called, even though it can still run nodes after this node.

Performance

In a single player game with all peripheral systems disabled, this node took ~1200 units to run on average.

Performance may vary for various reasons. These numbers are just for giving a sense of rough magnitude. Performance may differ from profiling in a test play when playing a published stage. First execution of a node within a node graph execution tend to take a bit longer.
