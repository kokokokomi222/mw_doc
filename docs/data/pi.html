Pi (π)

Returns 3.142.

Output Values

  • Pi (π) (Floating Point Numbers) - 3.142

Usage

This node returns 3.142, an approximate value of π. This is weird, because floats in Miliastra are 32-bit, so we can have more precision (roughly 7 decimal digits). You may want to define your own alternative composite node for this node (e.g. Degrees to Radians of 180, which returns 3.1415927), but in most cases the precision shouldn't matter too much (it's only off by ~0.013%).

You may want to use Degrees to Radians or Radians to Degrees instead if you are working with angles.

Performance

This node took ~2 units to run on average.

Performance may vary for various reasons. These numbers are just for giving a sense of rough magnitude. Performance may differ from profiling in a test play when playing a published stage. First execution of a node within a node graph execution tend to take a bit longer.
