Increase Skill Resource Amount

Modifying the resource amount of a skill will add an increase to the current value; this increase can be a negative number.

Input Parameters

  • Target Entity (Entity) - Active Character Entity
  • Skill Resource Config ID (Config ID) - Skill Resource Identifier
  • Increase Value (Floating Point Numbers) - The modified value is: original value + increase value.
