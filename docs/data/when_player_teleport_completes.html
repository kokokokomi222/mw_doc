When Player Teleport Completes

Triggers when a player completes teleportation.

Output Values

  • Player Entity (Entity) - Player entity that completed teleportation.
  • Player GUID (GUID) - GUID of the player entity.

Usage

Use this node to listen to event when player teleports. The event triggers after the teleport is finished. Typically, this node should be added to a node graph that is mounted on a player entity.

The official documentation claims that this node triggers "when a Player enters a Stage for the first time". However, this does not seem to be true. Use When Entity Is Created for the player entity to detect this event instead.

Notes

  • When a player revives, the player teleports to a revive point and thus this node is triggered.
  • This event can be forwarded by Forwarding Event, so this node could be on a node graph that is not mounted on a player entity.

Performance

This node took ~2 units to run on average.

Performance may vary for various reasons. These numbers are just for giving a sense of rough magnitude. Performance may differ from profiling in a test play when playing a published stage. First execution of a node within a node graph execution tend to take a bit longer.
