HP Loss

Directly cause the specified target to lose HP. Losing HP is not an attack, so it does not trigger attack-related events

Input Parameters

  • Target Entity (Entity) - Target that loses HP
  • HP Loss (Floating Point Numbers) - The amount of HP lost in this instance
  • Lethal (Boolean) - If set to False, this HP loss will leave the Target with at least 1 HP remaining
  • Can be blocked by invincibility (Boolean) - If set to True, and the Target is set to Invincible via Unit Status, HP loss has no effect
  • Can be Blocked by Locked HP? (Boolean) - If set to True, and the Target's HP is locked via Unit Status, HP loss has no effect
  • Damage Pop-Up Type (Enumeration) - No Pop-UpNormal Pop-UpCRIT Hit Pop-Up
