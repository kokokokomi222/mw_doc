Get Entity Type

Returns the type of an entity.

Input Parameters

  • Target Entity (Entity) - Entity to get the entity type of.

Output Values

  • Entity Type (Enumeration) - One of Stage, Object, Player, Character, or Creation.

Usage

Use this node to get the type of the entity.

Example

In this example, we get the type of the entity that is entering the collision trigger. If the entering entity is a character, then we hide the model of the entity.

Performance

This node took ~1 unit to run on average.

Performance may vary for various reasons. These numbers are just for giving a sense of rough magnitude. Performance may differ from profiling in a test play when playing a published stage. First execution of a node within a node graph execution tend to take a bit longer.
