Query Equipment Tag List

Gets the list of equipment tags from equipment instance.

Input Parameters

  • Equipment Index (Integer) - ID of the equipment. e.g. 3

Output Values

  • Tag List (Config ID List) - List of configuration ID of the equipment tags on the specified equipment.

Usage

This node lets you query the equipment tags from equipment. Equipment tags can be configured in the item detail panel of the equipment, under the "Interaction Settings" tab. This is currently the only way to set equipment tags.

Equipment tags can be configured to be shown in-game.

  • If Equipment Index is not an ID for equipment, this node returns the empty list.

Performance

This node took ~2 units to run on average.

Performance may vary for various reasons. These numbers are just for giving a sense of rough magnitude. Performance may differ from profiling in a test play when playing a published stage. First execution of a node within a node graph execution tend to take a bit longer.
