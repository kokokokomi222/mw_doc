Switch Main Camera Template

This server node is only available in Beyond Mode.

Switches players' camera.

Input Parameters

  • Target Player List (Entity List) - List of player entities.
  • Camera Template Name (String) - Name of the camera to switch to, as specified in the "Manage Main Camera" panel.

Usage

Various cameras with different perspective and configuration can be specified in the "Manage Main Camera" panel, which can be opened from the main menu. This node can be used to change the camera for specific players.

Example

In this example, we created a new camera named Koko's First Person Camera in the "Manage Main Camera" panel. In the node graph, all players change their camera to Koko's First Person Camera when a character selects the tab.

Notes

  • If Target Player List contains an entry that is not a player entity, this node raises "Entity does not exist" error (can be misleading, because the non-player entity may exist). This node ignores entries that are not player entities and successfully changes the camera for valid player entity entries.
  • If Camera Template Name is not a name for a camera, the camera changes to a follow character classic camera. In particular, it does not change to the default camera.

Performance

For a list of one player, this node took ~4 units to run on average.

Performance may vary for various reasons. These numbers are just for giving a sense of rough magnitude. Performance may differ from profiling in a test play when playing a published stage. First execution of a node within a node graph execution tend to take a bit longer.
