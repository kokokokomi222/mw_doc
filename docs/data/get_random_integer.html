Get Random Integer

Returns a random integer from range.

Input Parameters

  • Lower Limit (Integer) - Lower bound of the random range (range includes this value)
  • Upper Limit (Integer) - Upper bound of the random range (range includes this value)

Output Values

  • Result (Integer) - Randomly generated integer in the range

Usage

Use this node to get a random integer. The range is inclusive of both ends, so if Lower Limit is 3 and Upper Limit is 8 then the possible Result is 3, 4, 5, 6, 7, 8.

Although the official documentation does not state it, we can probably assume that it tries to approximate uniform distribution (i.e. equal probability of getting each possible output), but we can't say anything about the quality of the randomness.
